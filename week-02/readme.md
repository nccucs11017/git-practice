### nvm

全名為 Node Version Manager，顧名思義就是管理 Node.js 版本的工具，可以安裝多個版本的 Node.js，並在各版本間切換

### NPM

Node Package Manager，是 Node.js 其中一個，也是預設的套件管理系統，可以依據 package.json 安裝專案需要的套件，也可以用全域的方式把套件裝在電腦上
